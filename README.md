## 📦 安装

```bash
npm install corki-ui --save
```

```bash
yarn add corki-ui
```

## 🔨 示例

````jsx
import { Button } from 'corki-ui';

const App = () => (
  <>
    <Button type="primary">PRESS ME</Button>
  </>
);
````
